## 아이디어 요약
화재 피해액 관련 변수를 수집하여 데이터 전처리 후 신경망을 이용한 서울시 자치구별 위험도 분류 모델을 구축한다. 구축한 모델 중에서도 최고 성능을 보인 신경망 모델을 활용하여 데이터 분석 후 특정 자치구에 대한 피해액 수준 감소 솔루션을 제공한다.
